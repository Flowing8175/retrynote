# RetryNote — 상세 프로젝트 설명서

> AI 서비스 홍보 전략 수립용 참고 문서  
> 작성일: 2026-04-09

---

## 1. 프로젝트 개요

**RetryNote**는 학습자가 자신의 학습 자료를 업로드하면, AI가 자동으로 문제를 출제하고 채점까지 해주는 **AI 기반 개인 맞춤형 학습 플랫폼**입니다.

단순히 문제를 내고 끝나는 것이 아니라, 틀린 문제와 약한 개념을 추적해 그 개념만 집중적으로 다시 풀도록 유도하는 **적응형 복습 루프(Adaptive Retry Loop)**가 핵심입니다.

**서비스 URL**: https://retrynote.cloud

---

## 2. 해결하려는 문제

| 기존 학습 방식의 문제 | RetryNote의 해결책 |
|---|---|
| 공부한 내용을 어떻게 문제로 만들지 모름 | 자료만 올리면 AI가 자동 출제 |
| 어디서 틀렸는지 파악하기 어려움 | 오답노트가 자동 생성됨 |
| 틀린 개념을 다시 공부해도 또 잊어버림 | 약한 개념만 골라서 반복 출제 |
| 시험 환경 연습이 어려움 | 실제 시험처럼 일괄 채점하는 시험 모드 제공 |
| 학습 진도를 한눈에 파악하기 어려움 | 대시보드에서 정확도·취약 개념·코칭 메시지 제공 |

## 3. 핵심 기능

### 3.1 AI 문제 출제

- **자료 기반 출제**: PDF, 이미지, 텍스트 등 학습 자료를 업로드하면 AI가 해당 내용에서 문제 생성
- **자료 없는 출제**: 특정 주제나 과목명만 입력해도 문제 생성 가능
- **지원 문제 유형**: 객관식 / O×(참거짓) / 단답형 / 빈칸 채우기 / 서술형
- **출제 옵션**: 문제 수, 난이도, 유형, AI 모델 티어 선택 가능

### 3.2 AI 채점

- **즉시 채점 모드 (Normal)**: 문항 제출 즉시 정오답과 해설 제공
- **시험 모드 (Exam)**: 모든 문항을 풀고 나서 한 번에 채점 (실제 시험 환경 시뮬레이션)
- **유형별 채점 방식**:
  - 객관식·O×: 정규화 정확 일치
  - 단답형·빈칸: 후보 정답 일치 → AI 유연 채점 fallback
  - 서술형: GPT/Gemini AI 채점

### 3.3 오답노트 & 취약 개념 추적

- 틀리거나 건너뛴 모든 문제가 자동으로 **오답노트**에 기록
- 개념별로 `오답 횟수 / 부분점수 / 건너뜀 / 연속 실패` 지표를 추적하는 **WeakPoint 시스템**
- 오답노트에서 특정 개념을 선택해 재도전 퀴즈를 즉시 생성 가능

### 3.4 적응형 재도전 시스템 (Adaptive Retry)

- 대시보드가 WeakPoint 데이터를 분석해 **지금 가장 취약한 개념 TOP N**을 자동 추천
- "재도전" 기능으로 그 개념들만 집중 반복 출제
- 전통적인 SM-2 같은 시간 간격 알고리즘 대신 **개념 약점 기반 적응형 반복**을 사용
- 사용자가 더 잘 알수록 해당 개념은 추천에서 자동으로 내려감

### 3.5 학습 대시보드

- **정확도 현황**: 유형별·과목별·자료별 정확도 집계
- **취약 개념 목록**: WeakPoint 점수 상위 개념 표시
- **코칭 메시지**: AI가 학습 패턴을 분석해 맞춤형 코칭 문구 생성
- **재도전 추천**: 지금 당장 복습해야 할 개념 자동 선정

### 3.6 문서 처리 파이프라인

- **OCR**: 이미지·스캔 PDF에서 텍스트 추출
- **자동 파싱**: 구조화된 청크(chunk)로 분할, 임베딩 생성
- **벡터 검색**: pgvector 기반 의미 검색으로 관련 내용 탐색
- **파일 관리**: 폴더 구조, 이름 변경, 이동, 삭제, 재처리

### 3.7 개념 다이어그램

- 특정 개념에 대해 AI가 **시각적 개념 다이어그램(Mermaid)** 자동 생성
- 복잡한 개념 간 관계를 시각화해 이해도 향상

### 3.8 통합 검색

- 자료, 오답노트, 퀴즈 기록을 한 번에 검색
- 의미 기반(semantic) 벡터 검색 지원

```

     ↓

     ↓

  → AI 파싱/OCR/임베딩 → 퀴즈 출제 가능 상태 전환
     ↓
[퀴즈 생성 설정]
  · 자료 선택 / 문제 수 / 난이도 / 유형 / 모드(일반·시험) 선택
     ↓
[퀴즈 풀이]
  · 일반 모드: 문항별 즉시 피드백

     ↓

     ↓
[오답노트]
  · 오답 목록 필터링, 오류 유형 수정
  · 취약 개념 선택 → 재도전 퀴즈 생성
     ↓
[재도전]
  · AI 추천 개념 또는 사용자 선택 개념으로 새 퀴즈 생성
     ↓
[대시보드 복귀] → 반복
```

## 7. 데이터 모델 구조

### 핵심 도메인

```
User
 ├── Folder
 │    └── File
 │         ├── ParsedDocument
 │         │    └── DocumentChunk (임베딩 포함)
 │         └── ConceptDiagram
 │
 └── QuizSession
      ├── QuizSessionFile (자료 연결)
      ├── QuizItem (문항)
      │    ├── AnswerLog (답안 로그)
      │    └── Objection (이의제기)
      └── WeakPoint (개념별 약점 지표)

Subscription / CreditBalance / UsageRecord (요금제·크레딧)
```

### WeakPoint 모델 (핵심 차별점)

```python
# 개념별 약점 지표 - 적응형 재도전의 핵심
WeakPoint:
  concept: str           # 정규화된 개념 키
  wrong_count: int       # 오답 횟수
  partial_count: int     # 부분점수 횟수
  skip_count: int        # 건너뜀 횟수
  streak_wrong_count: int # 연속 실패 횟수
```

## 10. 주요 API 엔드포인트 목록

### 인증 (`/auth`)
- `POST /signup` — 회원가입
- `POST /login` — 로그인
- `POST /token/refresh` — 액세스 토큰 갱신
- `POST /verify-email` — 이메일 인증
- `POST /password/reset` — 비밀번호 재설정

### 자료 관리 (`/files`)
- `POST /files` — 파일 업로드
- `GET /files` — 파일 목록 조회
- `POST /folders` — 폴더 생성
- `PATCH /files/{id}` — 이름 변경·이동
- `DELETE /files/{id}` — 삭제
- `POST /files/{id}/retry` — 파싱 재시도
- `GET /files/{id}/download` — 다운로드

### 퀴즈 (`/quiz-sessions`)
- `POST /quiz-sessions` — 새 퀴즈 세션 생성
- `GET /quiz-sessions/{id}/items` — 문항 목록
- `POST /quiz-sessions/{id}/items/{item_id}/answer` — 답안 제출 (즉시 채점)
- `POST /quiz-sessions/{id}/submit` — 시험 모드 일괄 제출
- `GET /quiz-sessions/{id}/results` — 결과 조회
- `POST /quiz-sessions/{id}/items/{item_id}/objections` — 이의제기

### 오답노트 & 재도전 (`/wrong-notes`, `/retry-sets`)
- `GET /wrong-notes` — 오답노트 목록
- `PATCH /wrong-notes/{id}` — 오류 유형 수정
- `POST /retry-sets` — 재도전 세트 생성

### 대시보드·검색·다이어그램
- `GET /dashboard` — 학습 요약, 취약 개념, 코칭 메시지
- `GET /search` — 자료·오답·퀴즈 통합 검색
- `POST /diagrams` — 개념 다이어그램 생성
- `GET /diagrams/{concept}` — 다이어그램 조회

### 결제 (`/billing`)
- `GET /billing/usage` — 사용량 조회
- `POST /billing/checkout` — 구독 결제
- `POST /billing/cancel` — 구독 취소
- `POST /billing/webhook` — Paddle 웹훅

## 12. 홍보 시 강조할 핵심 메시지

1. **"공부한 자료가 그대로 문제가 됩니다"** — 내 노트에서 직접 출제되는 완전 맞춤형 퀴즈
2. **"틀린 것만 다시 풉니다"** — AI가 약점을 추적해 꼭 필요한 복습만 추천
3. **"실제 시험처럼 연습합니다"** — 시험 모드로 실전 감각 훈련
4. **"공부 방법을 AI가 코칭합니다"** — 대시보드의 맞춤형 코칭 메시지
5. **"이미지·스캔 자료도 됩니다"** — OCR로 손글씨·스캔 PDF까지 지원
